# FatehChaabat.github.io

Static portfolio website for Fateh Chaabat.

## Overview

- Built with a single `index.html` file and embedded CSS/JavaScript.
- Designed for GitHub Pages deployment from the repository root.

## Deployment

1. Push the repository to GitHub.
2. Enable GitHub Pages in repository settings.
3. Use the `main` branch and root folder as the publishing source.

## Notes

- The site includes a hero section, skills, experiences, education, and contact information.
- Replace `photo.jpeg` with your updated profile image if needed.
